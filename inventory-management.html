Inventory Management app

This is a simple inventory management app. User have different accessibility depend on the role.
If user is an admin, user is able to create another user and also add, edit and delete product details and transaction, means that normal user can just read information.
To test the app, simply login with below username and password and see the differenece.
Admin login = {username: admin, password:password}
User login = {username: user, password:password}}
